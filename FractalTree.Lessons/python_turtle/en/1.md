# Fractal Tree Turtle

```python
from turtle import *

fillcolor("magenta")
begin_fill()
speed(10)
up()
goto(0,500)
pensize(25)
down()
forward(250)
right(90)
forward(800)
right(90)
forward(350)
right(180)
circle(150,-180)
right(180)
forward(50)
left(90)
forward(500)
right(90)
forward(50)
end_fill()
color("purple")
forward(250)
right(90)
forward(800)
right(90)
forward(350)
right(180)
circle(150,-180)
right(180)
forward(50)
left(90)
forward(500)
right(90)
forward(50)
```
